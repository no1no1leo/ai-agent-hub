# CHANGELOG — Startup Repositioning

## Summary
This change set repositions AI Agent Hub from a crypto-forward marketplace narrative toward a startup-ready broker/orchestration narrative.

## Updated
- README.md
- QUICKSTART.md
- marketplace/api.py

## Added
- POSITIONING.md
- ROADMAP.md

## Intent
Keep the strongest technical primitives:
- bidding
- reputation
- winner selection
- observability

Reduce emphasis on:
- decentralization as headline message
- escrow as the main product identity
- marketplace-first storytelling

Increase emphasis on:
- agent routing
- broker architecture
- multi-agent orchestration
- startup-grade product direction
